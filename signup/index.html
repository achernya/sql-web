Sign up Just one easy command

Ready to get started with sql.mit.edu?

Head on over to an Athena computer and run 

$ add scripts; signup-sql
in a terminal. Then, follow the on screen instructions. That’s it, you’re all signed up!

You can find out your username and password by running 

$ cat ~/.my.cnf

Now that you’re registered, sign in to manage your databases!

You can also sign up group accounts (lockers) you have administrator privileges on. If you’re using Scripts Autoinstallers, they’ll automatically sign up for a sql.mit.edu account if needed.

Don’t have an Athena computer nearby?

athena.dialup has you covered. Log in with your MIT Athena Account and run the commands there. You can also use ssh username@athena.dialup.mit.edu from a Mac or Linux computer. If you’re using Windows, you can download and use PuTTY.

What if something goes wrong?

If you run into any problems, or have any questions, send an email to sql@mit.edu. We’re happy to help out!
